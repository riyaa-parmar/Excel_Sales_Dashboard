# 📊 Excel Sales Dashboard

An interactive sales dashboard built using Microsoft Excel to analyze business performance across products, regions, and salesmen.

---

## 🚀 Overview

This project provides a clear and interactive view of sales data using charts, KPIs, and slicers. It helps in understanding trends, identifying top-performing products, and evaluating regional and salesperson performance.

---

## 📌 Features

* 📊 KPI Metrics (Total Sales, Total Amount)
* 📅 Monthly Sales Trend Analysis
* 📦 Sales by Product
* 🌍 Sales by Region
* 👨‍💼 Sales by Salesman
* 🔝 Top & Bottom Performing Products
* 🎛 Interactive Slicers (Date, Region, Item)

---

## 🛠 Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Formatting

---

## 📂 Dataset

The dataset includes:

* Date
* Product (Item)
* Region
* Salesman
* Quantity
* Price
* Amount

---

## 📈 Key Insights

* Printer is the highest selling product
* East region generates the highest revenue
* Siddhu is the top-performing salesman
* Sales peak during February and September
